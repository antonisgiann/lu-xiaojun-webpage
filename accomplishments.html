Accomplishments
Lu Xiaojun is a four time world champion.
  1. Goyang,South Korea 77kg class 2009
  2. Paris,France 77kg class 2011
  3. Wroclaw,Poland 77kg class 2013
  4. Ashgabat,Turkmenistan 81kg class 2018
He took first place at London, olympic games 2012 winning both in snatch with 175kg and clean and jerk with 204kg
He took second place at Rio de Janeiro,Brazil olympic games 2016 finishing first in the snatch with 177kg and second in the clean and jerk with 202 kg
Lu Xiaojun in his weightlifting career has a complished a lot of records:
  1. Goyang,South Korea 2009 world champiship,a snatch world record with 174kg and a total world record with 378kg in the 77kg class
  2. London 2012 olympic games,snatch world record with 175kg and a total world record with 379kg in the 77kg class
  3. Wroclaw,Poland 2013 world championship,a snatch world record with 176kg and a total world record with 380kg in the 77kg class
  4. Rio de Janeiro,Brazil 2016 olympic games,a snatch world record with 177kg in the 77kg class
  5. Ashgabat,Turkmenistan 2018 world champioship,a snatch world record with 172kg and a total world record with 374kg in the 81kg class
